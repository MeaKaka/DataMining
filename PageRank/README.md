### PageRank


常见的求解特征向量的方法：**幂迭代法**<br>
```math
v_{k+1} = A v_k
```
当两者之差小于某个设定值时可以认为达到稳态了。<br>
但是在实际情况中需要考虑阻尼系数(α是阻尼系数):
```math
r = (1-α)G_mr + \frac{α}{n}r
```
G是稀疏矩阵，仍是稀疏矩阵，只是修改了特征向量的值。<br>
还有一个需要注意的部分是计算过程中矩阵的转置操作。

#### 特征向量与特征值：
如果矩阵对某一个向量或某些向量只发生伸缩变换，不对这些向量产生旋转的效果，那么这些向量就称为这个矩阵的特征向量，伸缩的比例就是特征值。<br>
特征向量的实质就是谱<br>

#### 矩阵的转置
```math
<x,A'y>=<Ax,y>
```
虽然上述式子没有什么意义<br>
```math
v_{k+1} = A v_k
```
首先，明确v是列向量，A与v相乘，显然是A的每一行乘上v的每一个元素v[i]得到新的v[i].v[i]可以看作是第i个状态的平稳分布，v[i]的值与其他状态指向i状态的概率有关，因此对A进行转置。

#### 可能问题
在PageRank过程中可能遇到的问题;
- 一是网页数较大，这时候可以利用稀疏矩阵，occ存储i，j以及value的值
- 二是如果页面A中，链接页面B的个数有不止一个的情况要如何处理，更改value的值，对应链接个数。
- 至于A，B之间的循环问题（成环）则不需要考虑，有环图不影响平稳的求解
- 使用幂法之前，要确保能够收敛

#### 总结
核心在于：
- 概率转移矩阵的转置
- ｖ的更新，特别是阻尼系数的考虑



###### 附:PCA主成分分析法
选取新的坐标轴，然后根据各个维度上的数据方差大小，决定保留哪些维度的数据，这样的做法就是主成分分析的核心思想。(方差越大的信息量越多)<br>
经过数学上的推导的，我们就可以知道，特征值对应的特征向量就是理想中想取得正确的坐标轴<br>
这个就是特征值和特征向量的一个实际应用：“得出使数据在各个维度区分度达到最大的坐标轴。”<br>
通常经过特征向量变换下的数据被称为变量的主成分，当前m个主成分累计的方差贡献率达到一个较高的百分数（如85%以上）的话，就保留着这m个主成分的数据。实现了对数据进行降维的目的。整个主成分分析的算法原理也就是这个。
